Intro to Arduino with the USDA Watershed Sesnsors board

Install Arduino Progaming Environment

https://www.arduino.cc/en/software and on both the following pages select "JUST DOWNLOAD" without  to avoid the random spam you will get when you join development communities.

Plug everything in and connect to a USB port on your computer:

Start and Configure Arduino IDE to connect to your board with the SparkFun ESP32 IoT ReadBoard

Grab/Download Introduction Code from Github:

https://github.com/WatershedMonitor/watershedmonitor.github.io

Unzip, and/or copy the folders that are in the "examples" directory into your Arduino sketches folder usually inside your Document's folder at:

C:\Users\{Your Windows User}\Documents\Arduino

Open the usda_station_intro.ino sketch in the Arduino IDE

Check Compile

Note that it fails with an error message stating that libraries are missing, and it highlights in the code the line where the library is being referenced.  

Install Required Libraries:

"Zanshin_BME680"

"SparkFun_Qwiic_Scale_NAU7802_Arduino_Library"

"ICM_20948"

"RadioLib"

Repeat for each using the following mouse driven workflows that show Zanshin_BME680 as an example

Press "Push" (compiles and loads) Open Serial Monitor, Select Baud Rate to 115200

If you don't have your baud rate set correctly, you will see the gibberish as shown below.

Congratulations!

You are now sensing and sending the data as displayed into the void through the LoRa radio via a 915mhz chirp. If you have a second board or another LoRa device, you can now listen to the frequency as configured in settings.h .
