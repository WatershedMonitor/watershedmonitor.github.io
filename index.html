Watershed Monitor Board

The Watershed Monitor Board is an Arduino-based project designed to monitor environmental conditions in watershed areas. This project utilizes a variety of sensors and components to collect and transmit data, ensuring comprehensive environmental monitoring.

Included Sensors and Components

BME680 Environmental Sensor

The BME680 sensor is an environmental sensor capable of measuring temperature, humidity, pressure, and volatile organic compounds (VOCs). This versatile sensor is ideal for monitoring the atmospheric conditions in watershed areas.

The sensor is integrated into the project using the Zanshin_BME680 library. For more details and to access the library, visit the Zanshin_BME680 GitHub page.

Digital Scale

The Digital Scale is used to measure the weight of collected samples or objects. This can be particularly useful in monitoring sediment levels or other materials in the watershed environment.

The scale is controlled using the SparkFun Qwiic Scale NAU7802 library. You can find more information and download the library from the SparkFun GitHub page.

Serial GPS Breakout

The Serial GPS Breakout provides accurate geolocation data, allowing the monitoring board to tag environmental data with precise location information. This is crucial for tracking the movement of water, sediments, and other materials within the watershed.

ESP32 Microprocessor

The ESP32 microprocessor is the brain of the Watershed Monitor Board. It features built-in BLE (Bluetooth Low Energy) and WiFi communications, enabling wireless data transmission and remote control of the board.

RF96 LoRa Long Range Radio

The RF96 LoRa Long Range Radio operates at 915MHz and allows the Watershed Monitor Board to transmit data over long distances, making it ideal for remote monitoring in vast watershed areas.
